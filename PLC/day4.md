근접 센서
검출체가 근접했을 때 건출대상 물체의 유무를판별하는 무접촉 방식의 센서

유도형 근접센서 : 물체의 자계 에너지를 이용하여 기계적 접촉없이 금속을 검출하는 센서
정전용량형 근접센서 : 센서와 물체 사이의 정전용량의 변화를 이용하여 물체를 검출하는 센서
흔히 용량형 센서라고 부르며 공기를 제외한 대부분의 물체를 검출가능

유도형 근접센서
유도형센서, Nomally Open, 3선식, NPN

광전센서
광을 발사하고 반사되어 되돌아오거나 차단되는 빛을 감지하는 센서
확산형 : 발광부에서 쏜 빛을 맞은 물체가 빛을 반사하는 것만으로 감지하여 검출
반사형 : 물체가 없을 땐 반사판에서 반사된 일정한 각도의 빛을 감지, 물체가 있을 땐 발광부에서 쏜빛이 물체에 막혀 반사되지 않음으로 검출
센서 맞은편에 반사판을 설치
투과형 : 발광부에서 쏜 빛을 반대쪽 수광부에서 감지하여 물체 유무 검출

### PLC 타이머
순시(즉시) : 전기적 힘이나 물리적 힘이 가해질때 바로 반응하는 것
한시(나중) : 전기적 힘이나 물리적 힘이 가해질때 즉시 반응하지 않는 것

코일에 전기를 공급하면 설정된 시간만큼 지연 후 코일 작동
전기를 차단하면 바로 접점이 바로 복귀
**시간 지연 기능**을 가진 릴레이
릴레이는 즉시 접점이 반전되지만 timer는 설정 시간 후 접점이 반전된다.

입력 "T0 K30"
T0 K30  T0타이머가 300ms후 작동
K가 붙지 않는 숫자는 16진수로 해석한다. K가 붙으면 10진수로 보인다.
