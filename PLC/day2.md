## Day2

### 릴레이 시뮬레이션
X1 -< M1 : 이 때 M1이 여자되고 내부 접점들이 반전된다.
M1 -< Y20 : 여자되며 연결이 된다.
!M1 -< Y21 : 여자되며 연결이 안된다.
END

여기에서 M1은 잠깐 X1의 상태를 저장하는 변수처럼 생성된되.
하지만 X1은 M1과 동일하게 왔다갔다 거리므로 추후 작업을 통해 기억을 시킬 수 있다.

### PLC CPU
가운데 레버를 통해 RESET-STOP-RUN CPU를 조절할 수 있다.
RUN : 시퀀스 프로그램 실행중
STOP 시퀀스 프로그램 정지 상태
RESET : 하드웨어 리셋(재시작) : 3초간 입력 유지 모드가 불이 꺼지고 다시 켜지는데 침착히 대응한다.



### PLC 자료 전송
write to PLC : PLC로 자료 전송
Read from PLC : PLC로 자료 수신

PLC에 전송할 때는 STOP이 되어야 한다.
다시 RUN으로 자동으로 바꾸거나 수동으로 바꿔줘야 한다.

F3번 모니터링 모드로 들어가게 되면 PLC와 프로그램 입력이 자동으로 연결된다.


### 자기유지회로
자기자신의값을 자기접점에 병렬로 연결한다.
X0 || M0 = M0
M0는 set이 되면 항상 1이 되는 회로이다.

따라서
(X0 || M0) && !X1 = M0
X0값이 1이면 X1이 0이므로 항상 M0는 1을 유지한다.
X1값을 1을 넣게되면 항상 M0는 0을 유지한다.
X0 시작버튼, X1 종료버튼


### 예시 

PB1 = L1
PB1 = L2
PB1 && PB2 = L3
PB1 || pb2 = L4

PB1누르면 L1, L2 점등 떼어도 점등 유지 PB3 누르면 전체 소등
Pb1, PB2 모두 누르면 L3 점등 떼어도 점등 상태 유지 PB3 누르면 모든 램프 소등
PB1, PB2 또는 누르면 L4 점등 떼어도 점등상태 유지 PB3를 누르면 모든 램프 소등


### 인터록 회로(InterLock) 선입 우선회로
X1 && !Y21 = Y20
X2 && !Y20 = Y21
먼저 눌린 입력이 먼저 작동한다.
ex) 승강기 버튼 처럼 두 버튼이 우선적으로 눌리지 않도록 하게된다.


자기유지 회로를 더한 버전

입력부
(X0 || M0) && !M1 && !X2 = M0
(X1 || M1) && M0 && !X2 = M1

출력부
M0 = Y20
M1 = Y21

X0, X1 둘 중 하나가 입력되었을 때 그 상태를 유지하고 다른 버튼은 적용되지 않는다.
X2 버튼을 통해 입력을 초기화한다.

### 공압(Pneumatic)(뉴 매릭)
작동부 : 유체의 에너지를 기계적 에너지로 변환 : 액추에이터
제어부 : 방향제어, 유량제어, 압력제어 등 : 밸브
압력발생부 : 전기 원동기 등 외부 동력에 의한 기계 에너지를 유체의 압력 에너지로 변환 : 전원

서비스 유닛(FBL 유닛) : Filter, Regulator, Lubricator
입력 : 수분 및 이물질 기체

에어필터 : 먼지 수분제거. 실린더 내부에 침입하면 먼지와 실린더에 녹이 슬게 된다.
레귤레이터 : 압력 조정. 
루브리케이터 : 윤활유 공급

출력 : 공압기기 활용에 적합한 기체 압축탱크에 보관

### 단동 실린더
스프링 때문에 최대 행정거리가 100mm로 제한된다.
실린더 헤드가 복귀할 때 큰 힘이 필요하지 않는 곳에 사용한다.
출력 접점을 1개만 사용하여 제어가 가능하다.

### 복동 실린더
행정 거리는 2000mm로 제한한다.
전후진 제어를 위해서는 두개의 접점이 필요하다.
압축 공기를 넣으면 전진, 압축공기를 빼면 후진한다.

### 방향 제어 밸브
공기의 방향을 조절한다.
내부에서 전기적신호, 버튼식 신호를 받아 계산한다.

### 유량 제어 밸브
공기의 유량을 제어한다.
한 방향으로 돌아오는 유량은 제어가 가능하나 반대 방향에서 들어오는 유량은 제어한다.
유량을 제어해서 실린더의 속도 등을 제어한다.


### 센서의 정의
대상을 감지 도는 측정하여 측정량을 전기적인 신호로 변환하는 장치
소리, 빛등을 가지하여 유용한 신호(디지털,아날로그)로 변환하는 장치

감지종류
