편솔 
자기 복귀 방식
전기를 주면 전진하지만 전기를 끊는 순간 스프링 힘에 의해 자리로 돌아간다.
안전이 중요하거나 전기가 나갔을 때 초기 위치로 복귀

미터아웃제어
공기는 막힘없이 들어가고 나가는 공기를 아주 빠르게 빼 버리도록 설정

전진 : 실린더 내부로 공기가 들어가서 피스톤 로드가 밖으로 밀려 나가는 것
작업물에 힘들가하거나 밀어낼 때
후진 : 피스톤 로드가 다시 실린더로 들어가는 것
