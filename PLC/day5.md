## Day5

### NPN, PNP

NPN 
부하 기준 (+) 입력기준 (-)
노이즈에 강하다. 전선비용이 감소한다.
0을 제어 한다. 의도치 않은 동작 유발이 가능하다.
일본 한국 기준

PNP
부하 기준 (-) 입력기준 (+)
노이즈에 약하다.
1을 제어한다. 비교적 안전하다.
유럽 미국 기준
개념적으로 편리하고 산업제어분야 선호된다.



### 적산 타이머
신호를 빼면 0으로 초기화된다.

그 후 일반 타이머 처럼 새로 켜지게 된다.

navigation - parameter - PLC parameter - Device

retensive timer 0k -> 1k
data register 13k -> 12k

타이머 시간 제한

Timer Limit Setting

### PLC 카운터
코일에 전기를 공급한뒤 빠진 값을 1로 체크한다.
32767회 까지 사용 가능하다.
카운터는 설정 횟수 도달 후 반전된다.

설정값 이상으로 카운트 되지 않음

### 카운터 자동 리셋
C0 = RST C0
카운터가 셋 되었을 경우 자동으로 자기자신을 리셋한다.

C0 = RST C0
X0 = C0 K3
(C0 | M0) & !X1 = M0
M0 = Y38

카운터의 경우 카운터가 셋 되었다는 것을 모든 프로그램이 다 알아야 하므로 가장 위에 자동 초기화를 적게된다.
하지만 프로그램에서 실행시 버튼이 눌림을 기억하므로 0으로 초기화가 되지 않는다.
SM402 = [RST C0]
SM402는 전원 공급(STOP -> RUN)시에 1스캔만 ON되게 만드는 기능이다.
전원을 끌때 까지 OFF가 된다.
전원 공급시 초기값이나
